### Dokimon Quest: Patch Notes
==============================
The xdelta patch included with this port makes the following modifications:

- Repack texture pages to 512x512 (from 2048x2048)
- Remove GMLive initialization scripts
- Set platform variables at runtime instead of compile time
- Implement frame buffering for the shader palette swap script

Made with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher), a GUI tool.
